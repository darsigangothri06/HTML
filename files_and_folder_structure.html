Project (Root) Folder:
This is the main folder for the project and contains all other files and folders.
On the web it's called root folder

HTML files:
These are all of the files that ends with .html.
These should be directly in the Project/Root folder and not in any sub folders.

CSS Folder:
All Cascading Style Sheet files go here.
They end in .css.
You can name the folder css.

Image Folder:
All images go here (.jpg, .png, .gif, etc).
You can name the folder images or img.

JavaScript Folder:
All JavaScript files go here.
They end in .js.
You can name the folder javascript or .js
