List Tags

ol tag:

Stands for ordered list
To define series of events that take place in some order.
Example making coffee, ingredients are:
  1. Milk
  2. Sugar
  3. Coffee Powder

If we want a. b. c. or A. B. C. or i. ii. iii. or I. II. III
Then we have to use spearate style.
If we want a,b,c then we have to write lower-alpha
If we want to write A, B, C then we have to write upper-alpha
If we want i, ii, iii then we have to write lower - roman
If we want I, II, III then we have to write upper-roman
If we did not mentioned anything then by default we get decimals (1,2,3...etc)

  3. Coffee
  4. Powder

ul tag:

Stands for unordered list.
To define series of events that take place where order is not important.
Example my hobbies:
  • Listening music
  • Browsing internet
  • Programming
Square
  • Listening music
  • Browsing internet
  • Programming
Circle
  • Listening music
  • Browsing internet
  • Programming

We also have different types in unordered list also like disc(default), square, circle etc

Definition List

When we are writing definitions line by line, we have so many definitions to write,
Then we use definition tag.
There are three tags which we use,

  1. DL (Definition List)
  2. DT (Definition Term)
  3. DD (definition Data or Description)

Nested List

List in List is called as Nested list.
Example:

  • American Breakfast Delicacies
    • Take Away Dishes
      • American Sausage
      • American Hot dog
        • Dark Sauce
        • Country special sauce
    • Dine in Dishes
      • Bacon and Fries
      • Fish and Chips
      • American Avacado Burger

We see another example for nested list.

  1. History
    • Mughal Empire

      1. Mongol Empire
      2. Russian Empire
  2. Mathematics
    1. Number System
      1. Ratio and Proportion
      2. Probability
    2. Differentiation
