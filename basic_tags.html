Different tags render different meaning.
Like,
h1 tag is for heading
p tag is for paragraph

Title tag:
Whatever is written in this tag comes up in the web page's tab.
Defines the title of the page.
Syntax: < open title tag > Text < close title tag >

Heading Tags:
Heading tags are from H1 to H6.
Search engines use the headings to index the structure and content of your web pages.
Users often skim a page by its headings. It is important to use headings to show the document structure.
H1 headings should be used for main headings, followed by h2 headings, then the h3, so on...
Note: Use HTML headings for headings only. Don't use headings to make big or bold.

The HTML abbr tag defines an abbreviation or an acronym, like "HTML", "CSS", "ASAP", "ATM".
Marking abbreviations can give useful inormation to browsers, translation systems and search engines.
